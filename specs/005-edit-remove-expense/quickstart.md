# Quickstart: Edit and Remove

## Deleting an Expense

1. Navigate to **Dashboard**.
2. Locate an expense in the **Recent Expenses** list.
3. Click the **Delete** (Trash icon/Text) button.
4. **Verify**: Item disappears immediately.

## Editing an Expense

1. Navigate to **Dashboard**.
2. Click **Edit** (Pencil icon/Text) on an expense.
3. **Verify**: Form opens with pre-filled data.
4. Change **Amount** to `999`.
5. Click **Save**.
6. **Verify**: Dashboard reappears, item shows `999`.

## Canceling Edit

1. Click **Edit**.
2. Click **Cancel** (or secondary button).
3. **Verify**: Dashboard reappears, no changes saved.
